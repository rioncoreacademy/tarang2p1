How it works
Two ways to run — pick what suits you
☁️ Option A — Browser only (GitHub Codespaces, no install)
Works on any device. Uses your free GitHub account. No Docker needed.
1

Login with GitHub

Students sign in with their free GitHub account — no new account registration needed.

2

Open Codespace

Click "Open in Codespace". GitHub starts a full Linux container with all tools pre-installed.

3

Start the Lab

A full Linux desktop opens in the browser. Write code, simulate, view waveforms — all from browser.

4

Submit via Git

Students push their completed lab to GitHub. Instructor reviews via pull requests.

🖥️ Option B — Local install (Docker Desktop, works offline)
Runs entirely on your PC/laptop. No internet needed after setup. Windows, Mac, Linux.
1

Install Docker Desktop

Download and install Docker Desktop for Windows or Mac from docker.com — one-time setup, takes ~5 minutes.

2

Pull the Lab Image

Open a terminal and run:
docker pull ghcr.io/rioncoreacademy/tarang2-dp1:latest

3

Start the Lab

Run with the command your instructor gives you (includes CLASS_TOKEN). Then open http://localhost:6080 in your browser.

4

Stop When Done

docker stop tarang2-dp1-lab
All your work is saved automatically via Git.
